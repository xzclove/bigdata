##**config**
>  config 是hadoop 相关的常用配置参数。


- FullyDistributed-HA  ，完全分布式，高可用环境配置修改文件
- FullyDistributed     ，完全分布式，非可用环境配置修改文件
- Pseudodistributed    ，伪分布式环境配置修改文件
- hbase-config         ，Hbase需要修改环境文件
- hbase-mapreduce      ，hbase运行mr的配置文件
